Tổng Quan Hệ Thống

MamaCore là một website dịch vụ chăm sóc sức khỏe mẹ và bé chuyên nghiệp, cung cấp các dịch vụ khám thai, chăm sóc sau sinh, khám sơ sinh, tư vấn nuôi dạy con, và nhận đặt lịch tư vấn trực tuyến 24/7.

Thông Tin:

Hệ thống được xây dựng với PHP,HTML,CSS, MySQL, Bootstrap 5 , Script,XAMPP cung cấp giao diện thân thiện và chức năng quản lý thông tin đặt lịch.

Đặc Điểm Chính

  • Giao diện hiện đại: Sử dụng Bootstrap và CSS để tạo giao diện đẹp mắt
  • Quản lý đặt lịch: Hệ thống đặt lịch khám, dịch vụ online
  • Bảng điều khiển admin: Thống kê, quản lý đầy đủ các thông tin khách hàng đặt dịch vụ,yêu cầu liên hệ dịch vụ quan tâm
  • Xác thực an toàn: Sử dụng password hashing với PHP password_verify()
  • Cơ sở dữ liệu bảo mật: Sử dụng PDO với prepared statements để phòng chống SQL Injection
  • Responsive Design: Tương thích với mọi thiết bị (desktop, tablet, mobile)

Yêu Cầu Hệ Thống

Yêu Cầu Máy Chủ
  • Phần mềm máy chủ: XAMPP (khuyến nghị)
  • PHP 7.4 trở lên (đã tích hợp sẵn trong XAMPP)
  • MySQL 5.7 trở lên / MySQL 8.0 (XAMPP)
  • Apache Web Server (có sẵn trong XAMPP)

                                    return $_SESSION['user_data_complete'];

                                } catch (Exception $e) {

                                }

6. Kết Nối Database (database.php)

PDO Connection
  • Sử dụng PDO (PHP Data Objects) cho bảo mật
  • Prepared statements để chống SQL Injection
  • Error mode: PDO::ERRMODE_EXCEPTION
  • Charset: utf8 để hỗ trợ tiếng Việt
               
// Set timezone for Vietnam
date_default_timezone_set('Asia/Ho_Chi_Minh');

class Database {
    private $host = 'localhost';
    private $db_name = 'csmvb';
    private $username = 'root';
    private $password = '';
    private $conn = null;

    public function connect() {
        try {
            $this->conn = new PDO(
                "mysql:host=" . $this->host . ";dbname=" . $this->db_name . ";charset=utf8",
                $this->username,
                $this->password,
                [
                    PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
                    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
                    PDO::ATTR_EMULATE_PREPARES => false
                ]
            );
        } catch(PDOException $e) {
            die("Connection failed: " . $e->getMessage());
        }

        return $this->conn;
    }

    public function getConnection() {
        if ($this->conn === null) {
            $this->connect();
        }
        return $this->conn;
    }
}

REATE DATABASE csmvb CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE csmvb;
CREATE TABLE `bookings` (
  `id` int(10) UNSIGNED NOT NULL,
  `service_name` varchar(255) NOT NULL COMMENT 'Tên dịch vụ',
  `service_price` varchar(100) DEFAULT NULL COMMENT 'Giá dịch vụ',
  `customer_name` varchar(255) NOT NULL COMMENT 'Tên khách hàng',
  `phone` varchar(20) NOT NULL COMMENT 'Số điện thoại',
  `email` varchar(255) DEFAULT NULL COMMENT 'Email khách hàng',
  `note` text DEFAULT NULL COMMENT 'Ghi chú của khách',
  `status` enum('pending','contacted','confirmed','completed','cancelled') NOT NULL DEFAULT 'pending',
  `created_at` datetime NOT NULL COMMENT 'Thời gian tạo',
  `updated_at` datetime DEFAULT NULL COMMENT 'Thời gian cập nhật'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;

CREATE TABLE `users` (
  `id` int(11) NOT NULL,
  `username` varchar(50) NOT NULL,
  `email` varchar(100) NOT NULL,
  `password` varchar(255) NOT NULL,
  `full_name` varchar(100) NOT NULL,
  `phone` varchar(20) DEFAULT NULL,
  `address` text DEFAULT NULL,
  `role` enum('customer','admin','staff') DEFAULT 'customer',
  `status` enum('active','inactive','banned') DEFAULT 'active',
  `created_at` timestamp NULL DEFAULT current_timestamp(),
  `updated_at` timestamp NULL DEFAULT current_timestamp() ON UPDATE current_timestamp()
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;

Hướng Dẫn Sử Dụng

🧑‍💼 Hướng Dẫn Cho Khách Hàng

1. Xem Trang Chủ

1
Truy Cập Website

Mở trình duyệt và truy cập trang chủ MamaCore tại: http://localhost/CSMVB/index.php

2
Xem Thông Tin

Xem các thông tin giới thiệu, tại sao chọn MamaCore, dịch vụ chính, và thống kê trên trang chủ

3
Điều Hướng

Sử dụng menu thanh điều hướng để đi đến các trang khác

2. Xem Danh Sách Dịch Vụ

1
Truy Cập Trang Dịch Vụ

Nhấn "Dịch Vụ" trong menu thanh điều hướng hoặc "Xem Tất Cả Dịch Vụ" trên trang chủ

2
Xem Chi Tiết

Xem mô tả chi tiết, thành phần bao gồm và giá cả của từng dịch vụ

3
Chọn Dịch Vụ

Nhấn "Đặt Lịch Ngay" để đặt dịch vụ mà bạn quan tâm

3. Đặt Lịch Khám

1
Mở Modal Đặt Lịch

Nhấn "Đặt Lịch Ngay" từ trang dịch vụ hoặc trang liên hệ

2
Điền Thông Tin
  • Dịch vụ: Được tự động điền từ dịch vụ bạn chọn
  • Giá dịch vụ: Được tự động điền
  • Họ và tên: Nhập tên của bạn (*bắt buộc)
  • Số điện thoại: Nhập SĐT liên lạc (*bắt buộc)
  • Email: Nhập email (tùy chọn)
  • Ghi chú: Thêm yêu cầu đặc biệt nếu có
3
Gửi Đặt Lịch

Nhấn "Đặt Lịch" để gửi yêu cầu. Một thông báo sẽ hiển thị xác nhận đặt lịch thành công.

4
Xác Nhận

Chúng tôi sẽ liên hệ với bạn qua SĐT hoặc email trong 24 giờ để xác nhận và sắp xếp lịch khám.

4. Liên Hệ Trực Tiếp

1
Truy Cập Trang Liên Hệ

Nhấn "Liên Hệ" trong menu để đi đến trang liên hệ

2
Xem Thông Tin Liên Hệ

Xem địa chỉ, số điện thoại, email, giờ làm việc và vị trí chi nhánh trên bản đồ

3
Gửi Tin Nhắn

Điền biểu mẫu liên hệ với dịch vụ quan tâm, thông tin cá nhân và nội dung tin nhắn

4
Nhận Phản Hồi

Chúng tôi sẽ phản hồi lại trong thời gian sớm nhất

👨‍💼 Hướng Dẫn Cho Quản Trị Viên

1. Đăng Nhập Admin

1
Mở Trang Đăng Nhập

Truy cập: http://localhost/CSMVB/admin/login.php

2
Nhập Thông Tin
  • Username: admin
  • Password: 123456 (hoặc password đã thiết lập)
3
Đăng Nhập

Nhấn nút "Đăng nhập" để vào dashboard

2. Quản Lý Đặt Lịch

1
Xem Danh Sách

Dashboard sẽ hiển thị danh sách tất cả các đặt lịch với thông tin chi tiết

2
Tìm Kiếm

Nhập tên khách hàng, email hoặc số điện thoại trong ô tìm kiếm để lọc đặt lịch

3
Lọc Theo Trạng Thái
  • Chờ xác nhận (pending)
  • Đã xác nhận (confirmed)
  • Hoàn thành (completed)
  • Huỷ (cancelled)
4
Cập Nhật Trạng Thái

Chọn trạng thái mới từ dropdown ở cột "Trạng thái". Hệ thống sẽ cập nhật tự động.

5
Xem Chi Tiết

Nhấn biểu tượng mắt để xem chi tiết đầy đủ của đặt lịch

6
Xóa Đặt Lịch

Nhấn biểu tượng thùng rác để xóa đặt lịch. Xác nhận khi được hỏi.

3. Xem Thống Kê

Các Thống Kê Hiển Thị
  • Tổng đặt lịch: Số lượng tất cả các đặt lịch
  • Chờ xác nhận: Số lượng đặt lịch chưa được xác nhận
  • Đã xác nhận: Số lượng đặt lịch đã được xác nhận
  • Hoàn thành: Số lượng đặt lịch đã hoàn thành

4. Đăng Xuất

1
Nhấn Đăng Xuất

Nhấn nút "Đăng xuất" ở góc trên bên phải của trang

2
Xác Nhận

Bạn sẽ được chuyển về trang đăng nhập
